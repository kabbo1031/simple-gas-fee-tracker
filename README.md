# Simple Gas Fee Tracker

This is a simple blockchain project for beginners.

This project helps you:
- Read current gas price
- Show slow, normal, and fast gas fees
- Learn how transaction fees work
- Learn how blockchain networks handle traffic

This project is good for:
- Web3 learners
- Blockchain students
- Crypto developers who want practice

---

## What You Need

- Web browser
- Internet connection
- Basic JavaScript knowledge

---

## How To Run

1. Download this repository
2. Open the folder
3. Open `index.html` in your browser
4. Click "Load Gas Fees"

---

## Project Files

- `index.html` = Main page  
- `app.js` = Main logic  
- `config.js` = Settings  
- `gas.js` = Gas reader  
- `provider.js` = Blockchain provider  
- `ui.js` = Screen updates  
- `utils.js` = Helpers  
- `style.css` = Styles  
- `logger.js` = Logs  
- `package.json` = Project info
